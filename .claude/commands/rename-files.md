# Rename Files by Content
---
description: 파일 내용을 분석하여 적절한 파일명으로 변경합니다
---

PDF 예시: "Arxiv 논문 파일 이름을 내용 기반으로 한글명으로 변경"

## 입력
- $DIR: 대상 디렉토리 경로
- $LANGUAGE: 파일명 언어 (ko/en, 기본: ko)

## 수행 단계

1. **파일 목록 조회**
   ```bash
   ls -la $DIR
   ```

2. **각 파일 분석**
   - PDF: 첫 페이지 텍스트 추출
   - 텍스트: 첫 몇 줄 읽기
   - 이미지: EXIF 정보 또는 내용 분석

3. **새 파일명 제안**
   - 내용을 잘 설명하는 이름으로
   - 특수문자 제거
   - 날짜 정보 포함 (있다면)

4. **사용자 확인**
   변경 전 목록 보여주고 확인 요청:
   ```
   original.pdf → 딥러닝_기반_추천시스템_설계.pdf
   2401.12345.pdf → Transformer_아키텍처_최적화_연구.pdf
   ```

5. **파일명 변경 실행**
   ```bash
   mv "$OLD_NAME" "$NEW_NAME"
   ```
