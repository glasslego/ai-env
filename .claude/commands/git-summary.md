# Summarize Git History
---
description: 최근 Git 변경 이력을 분석하고 요약합니다
---

## 입력 (선택)
- $DAYS: 조회할 기간 (기본: 7일)
- $AUTHOR: 특정 작성자 필터

## 수행 단계

1. **Git 이력 조회**
   ```bash
   git log --since="$DAYS days ago" --pretty=format:"%h|%ad|%an|%s" --date=short
   ```

2. **분석**
   - 날짜별 커밋 그룹핑
   - 주요 변경 내용 파악
   - 파일별 변경 빈도 확인

3. **요약 보고서 생성**
   - 📅 일자별 주요 변경사항
   - 📁 가장 많이 변경된 파일
   - 👤 기여자별 활동 (팀 프로젝트인 경우)
   - 🎯 주요 피처/버그픽스 요약

4. **변경 의도 추론**
   - 커밋 메시지와 코드 변경을 분석
   - "이 사람이 도대체 뭘 했는지" 추론
