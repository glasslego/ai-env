비트코인 자동매매에서 사용되는 퀀트 전략들을 기술적으로 심층 분석해줘.

반드시 포함할 내용:
1. 전략 유형별 수학적 모델과 Python 구현 가이드
   - 모멘텀 (RSI, MACD, Bollinger Bands)
   - 평균회귀 (Z-score, Cointegration)
   - 차익거래 (삼각차익, 거래소간 차익)
   - 마켓메이킹 (bid-ask spread)
2. 백테스트 vs 라이브 성과 괴리 문제
   - Overfitting 방지 기법 (Walk-forward, Cross-validation)
   - Look-ahead bias, Survivorship bias
3. ML/DL 기반 전략의 실전 성과
   - LSTM, Transformer 기반 가격 예측 논문 리뷰
   - 강화학습 (DRL) 트레이딩 에이전트
4. 리스크 관리
   - Kelly Criterion, VaR, CVaR
   - 포지션 사이징, 드로다운 제어

출력 형식: 마크다운, 논문 인용 포함
